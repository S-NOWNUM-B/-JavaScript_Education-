## О проекте

> **От теории к практике: создание браузерных игр и интерактивных приложений**

Этот репозиторий служит для отработки фундаментальных навыков JavaScript без использования фреймворков. Проекты сосредоточены на манипуляциях с DOM-деревом, обработке пользовательских событий, работе с таймерами и базовыми структурами данных (массивы, объекты).

## Структура проекта

<div align="center">

| Папка | Уровень | Описание |
|:------|:--------|:---------|
| **Guess_number** | Начальный | Логическая игра с вводом данных, проверкой условий и изменением CSS-стилей через JS. |
| **Rock-paper-scissors** | Начальный | Классическая игра. Работа с `Math.random()`, условными операторами и сравнением. |
| **Quiz** | Средний | Викторина. Динамическая генерация HTML из JS-объектов, подсчет очков и обработка форм. |
| **Hangman** | Продвинутый | Игра "Виселица". Работа с событиями клавиатуры, строковыми методами и массивами, рисование (Canvas/SVG). |

</div>
