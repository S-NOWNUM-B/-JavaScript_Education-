## О проекте

> **От теории к практике: создание браузерных игр и интерактивных приложений**

Этот репозиторий служит для отработки фундаментальных навыков JavaScript без использования фреймворков. Проекты сосредоточены на манипуляциях с DOM-деревом, обработке пользовательских событий, работе с таймерами и базовыми структурами данных (массивы, объекты).
